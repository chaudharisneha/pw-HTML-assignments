HTML (HyperText Markup Language)

    Tech Stack Name Primary Use Cases Key Features / Benefits
    HTML
    • Structuring web content
    • Creating web pages
    • Embedding multimedia (images, videos)
    • Linking documents
    • Fundamental building block of web pages
    • Defines the structure and layout
    • Supports multimedia elements
    • Semantics: Provides meaning to content
    • Accessibility: Enhances web accessibility

    CSS (Cascading Style Sheets)

    Tech Stack Name Primary Use Cases Key Features / Benefits
    CSS
    • Styling web pages
    • Layout control
    • Animation
    • Visual effects
    • Separation of content and design
    • Responsive design: Adapts to different devices
    • Flexbox and Grid: Powerful layout systems
    • Customizable: Variables and pre-processors
    • Cross-browser compatibility

    JavaScript

    Tech Stack Name Primary Use Cases Key Features / Benefits
    JavaScript
    • Adding interactivity to web pages
    • Form validation
    • Dynamic content
    • Client-side scripting: Runs in the browser
    • Versatile: Can be used for both front-end and back-end
    • Asynchronous programming: Promises and async/await
    • Extensive ecosystem: Libraries and frameworks
    • Event-driven programming
