Ordered List of HTML Tags

  1. body: Contains the content of the HTML document
  2. h1 to h6: Represent heading of different level highest to lowest.
  3. p: Represents a paragraph of text.
  4. a: Defines a hyperlink, which is used to link from one page to another.
  5. img: Embeds an image into the HTML document.
  6. ul: Defines an unordered list of items.
  7. ol: Defines an ordered list of items.
  8. li: Defines a list item within an unordered or ordered list.
  9. div: Defines a division or section in an HTML document
  10. span: Used to group inline-elements in a document.
  11. table: Defines a table in an HTML document.
  12. tr: Defines a row in a table.
  13. td: Defines a cell in a table row.
  14. form: Defines an HTML form for user input.
  15. button: Defines a clickable button.
